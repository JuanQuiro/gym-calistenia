# Plan Maestro: Proyecto Mister Venezuela (1 Año)

**Ubicación:** `/home/guest/Documents/gym-calistenia`
**Meta:** 50kg -> 80kg en 12 meses.
**Enfoque:** Hipertrofia Agresiva + Estética + Farmacología (Bajo responsabilidad del usuario).

## Cronograma Anual (La Carrera contra el Reloj)

### Fase 1: La Base (Meses 1-3)
*Objetivo: Llegar a 60kg (+10kg).*
- **Nutrición:** Superávit agresivo (3000+ kcal). Tu estómago se expandirá.
- **Entreno:** Adaptación anatómica y fuerza base.
- **Química:** **NO USAR AÚN.**
    - *Razón:* Si usas química con 50kg, tus tendones se romperán al subir la fuerza de golpe. Necesitas 3 meses para fortalecer articulaciones.
    - *Acción:* Analíticas de sangre completas (Perfil hormonal, hepático, lipídico) para tener una línea base.

### Fase 2: El Despegue (Meses 4-6)
*Objetivo: Llegar a 70kg (+10kg).*
- **Nutrición:** Ajuste a 3500 kcal.
- **Entreno:** Hipertrofia pura (Rango 8-12 reps).
- **Química (Si decides proceder):** Introducción teórica de compuestos (Testosterona base).
    - *Nota:* Primobolan es suave pero caro y requiere mucho volumen de aceite. Testosterona es la base de todo ciclo.

### Fase 3: Consolidación y Calidad (Meses 7-9)
*Objetivo: Llegar a 75-78kg.*
- El peso subirá más lento. Aquí buscamos densidad, no solo agua y grasa.
- Enfoque en hombros y amplitud dorsal.

### Fase 4: Puesta a Punto (Meses 10-12)
*Objetivo: 80kg Magros (Look de Tarima).*
- Recorte de grasa si es necesario (aunque siendo ectomorfo, probablemente te mantengas seco).
- Posing y preparación para casting.
